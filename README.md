🚀 Mahnoor AI Hub - GenAI Cohort 3 Project
Welcome to the Mahnoor AI Hub! This project was developed as part of the HEC Generative AI Training (Cohort 3). It is a specialized AI assistant designed to provide a smooth starting point for beginners exploring the world of Artificial Intelligence.

🌟 Features
Interactive Chatbot: A rule-based AI assistant built to answer introductory queries.

Streamlit Integration: A clean and user-friendly web interface for seamless interaction.

Optimized Code: The backend logic has been refactored for efficiency and faster response times.

Cloud Hosted: Fully deployed and accessible via Hugging Face Spaces.

🛠️ Tech Stack
Language: Python

Framework: Streamlit

Deployment: Hugging Face Spaces

Version Control: GitHub
